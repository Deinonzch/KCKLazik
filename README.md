KCKPoszukiwaczSkarbow
=====================

Projekt na KCK

**Krótki opisy projektu:** Poszukiwacz skarbu przemieszcza się po mapie 2d za pomocą poleceń użytkownika i poszukuje skarbów na określonych polach mapy.

**Wykorzystujemy SFML 2.1**
